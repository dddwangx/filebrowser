核心功能
========
1. 用户管理
   - 自动生成管理员和普通用户
   - 随机安全密码
   - JSON和文本格式导出

2. Logo轮播
   - 15张自然风景和动物图片
   - 每5秒自动切换
   - 淡入淡出动画

3. 登录页美化
   - 彩色渐变背景（5色）
   - 呼吸动画效果
   - 彩色渐变文字（加粗）
   - 毛玻璃登录框
   - 响应式设计

技术栈
======
- 后端: Filebrowser (Go)
- 前端: HTML5 + CSS3 + JavaScript
- 容器: Docker
- 脚本: Python 3 + Bash
- 图片API: Picsum Photos

快速命令
========
# 预览登录页
./scripts/preview_login.sh

# 查看管理员密码
cat config/login_credentials.txt | head -n 10

# 重新生成用户
python3 scripts/generate_users.py

# 重新下载图片
python3 scripts/download_images_improved.py

# 启动服务（需要docker-compose）
docker-compose up -d
